# Minigame de Adivinhação em Python
Minigame interativo em Python que mistura desafios de lógica, matemática e adivinhação com uma narrativa dinâmica. A experiência muda conforme os erros do jogador, através de um sistema de “sanidade” que altera falas e efeitos no terminal.

# Sobre o projeto
Durante o jogo, o usuário interage com a personagem Banbaleena, cuja personalidade muda conforme o desempenho do jogador. Um sistema de sanidade influencia diretamente a experiência, alterando falas, velocidade do texto e até distorcendo mensagens, criando uma atmosfera progressivamente mais imersiva.

# Funcionalidades
- Desafios de lógica e matemática
- Sistema de sanidade dinâmica
- Texto com efeito de digitação
- Alteração de humor da personagem
- Elementos narrativos e psicológicos

# Tecnologias utilizadas
- Python
- Módulos: time, os, random

# Como executar
1. Clone o repositório:
  git clone https://github.com/roodrigueslxz/adivinhacao_python.git
2. Acesse a pasta do projeto:
  cd adivinhacao_python
3. Execute o jogo:
  python minigame.v2.py

# Objetivo
Praticar lógica de programação, estruturas condicionais, loops, funções e tratamento de fluxo, além de explorar a criação de experiências interativas em terminal.
